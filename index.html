Privacy Policy — Funnelish Rich Paste

Last updated: March 23, 2026

Funnelish Rich Paste is a Chrome extension that pastes AI-generated rich text into the Funnelish editor with formatting preserved.

Data Collection

This extension does not collect, transmit, store, or share any personal data. All text processed stays within your browser and is never sent to any external server.

Permissions Used

activeTab — To interact with the Funnelish editor tab only when you trigger the paste action.

scripting — To inject the content script into the Funnelish editor to insert your formatted content.

Host access (editor.v2.funnelish.com) — The extension runs only on the Funnelish editor domain.

Third Parties

No analytics, advertising, or tracking. No data is shared with or sold to any third party.
