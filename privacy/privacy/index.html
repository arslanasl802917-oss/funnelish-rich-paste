Privacy Policy — Funnelish Rich Paste Last updated: March 23, 2026 Funnelish Rich Paste ("the Extension") is a Chrome browser extension that enables users to paste AI-generated rich text into the Funnelish page editor with formatting (bold, italic, spacing) preserved. DATA COLLECTION This extension does not collect, transmit, store, or share any personal data. It does not access your browsing history, identity, financial information, communications, or any other personal information. The extension operates entirely on your local device. All text processed by the extension stays within your browser and is never sent to any external server or third party. PERMISSIONS The extension requires the following permissions solely to perform its core function: - activeTab: To interact with the Funnelish editor tab that is currently open, only when you explicitly trigger the paste action. - scripting: To inject the content script into the Funnelish editor page so it can locate the active text block and insert your formatted content. - Host access (editor.v2.funnelish.com): The extension operates only on the Funnelish editor domain. It does not run on any other website. No data leaves your browser at any point. THIRD PARTIES This extension does not use analytics, advertising networks, tracking pixels, or any third-party services. No data is shared with or sold to third parties. CHANGES TO THIS POLICY If this policy is updated, the "Last updated" date above will be revised. Continued use of the extension after changes constitutes acceptance.
